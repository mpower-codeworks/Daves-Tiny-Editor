DTE  is based on the  tiny.asm  windows  application
by Dave Plummer.

Open a file by drag / drop onto the .exe or from the
command line> dte filename. This style was chosen to
avoid having a save filename:  dialog or text input.

The window class name is "." to save bytes and since
we already have the file name we can use that as the
caption.   It costs almost nothing to do so and also
helps a lot with the UI.  The modified file * seemed
like a natural fit.

File: Save was appended to the window system menu to
avoid adding a menu bar.

New for version 2.0 -- Rich Edit control integration
allows loading Courier mono-spaced font. DTE can now
be used to edit itself (painfully). 

The  RICHEDIT50W  control handles most of the  heavy
lifting.  Typing, backspace, mouse interaction, text
highlighting, and memory handling come pre-packaged.
The text-wrap default style is used.
